## Express.js Project | Testing custom HTTP Error/Sending Static files over Express.js

# Dependencies

All of the node_modules (dependencies for the project) are
pre-installed, however the project requires:

Node.js (on macOS it can be installed with brew)
nodemon (npm install -g nodemon)

# Run

To run the server, enter a terminal and run:
nodemon index.js

# Contributions

If you would like to contribute to this project,
please make sure to make a pull request with:

A meaningful description of what the changes are making.

# When contributing
Please make sure that you are not commiting any unnecessary
changes. These include white spaces, new lines, and code which makes the server run slower.
